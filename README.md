# Personal Portfolio Website

This project is developed as part of the **Future Interns Full Stack Web Development Internship – Task 1**.

## Project Overview

The Personal Portfolio Website is designed to showcase my skills, projects, and professional background. It acts as an online resume where visitors can learn more about my technical abilities, experience, and work.

The website includes multiple sections such as About, Skills, Projects, and Contact, allowing users to easily explore my profile and connect with me.

## Features

• Interactive and responsive portfolio layout  
• About section introducing my background  
• Skills section displaying technical skills  
• Projects section highlighting academic and technical projects  
• Resume download option  
• Direct links to GitHub and LinkedIn profiles  
• Contact form for communication  

## Technologies Used

HTML5  
CSS3  
JavaScript  

## Projects Included

1. **Sentiment Analysis Dashboard**
   - Built using Streamlit and machine learning
   - Analyzes public sentiment from user comments

2. **Medical Image Classification**
   - Uses Vision Transformers and CNN models
   - Applied in medical image analysis

3. **Hospital Management System**
   - A mini project to manage hospital operations such as patient details, doctor records, and appointments

4. **Coffee Growers Chatbot**
   - An AI chatbot designed to assist coffee growers with information related to coffee cultivation
